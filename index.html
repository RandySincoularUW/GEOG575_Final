About This Map

This map is my Final Project for GEOG 575

The Leaflet library, was used to create this map that combines 2016 parcel data and a selection of 40 Cottage Grove businesses. The parcels and business features have been converted to geoJSON format in a WGS 1984 Geographic Coordinate System. This web application includes a Mapzen geocoder and an optional Weather Radar layer from Iowa State University.

Location

Cottage Grove, Wisconsin 53527
43.0761, -89.1996

Purpose

This map is intended to be used for situational awareness in the event of an emergency or disaster.

Using this Web Map

- Hover over an icon to get business name and phone number

- Left click on a parcel to get Property Owner and Billing Address

- Left click on layer icon to turn on/off Weather Radar and Parcels

- Left click the magnifying glass icon to enter an address and zoom to that location
